# Proyecto 1 – Gestión de Archivos en Google Cloud

## Descripción
Sistema web que permite subir, listar y descargar archivos.
Cada archivo subido activa automáticamente una Cloud Function.

## Tecnologías
- Node.js
- Express
- Multer
- Google Cloud Storage
- Cloud Functions
- Compute Engine
- Nginx

## Instalación
cd app
npm install
node index.js
